# Marketing Skills

> 33 marketing skills for AI agents — SEO, GEO, conversion optimization, copywriting, paid ads, growth, and revenue operations.

Give any AI agent marketing superpowers. Works with **Claude Code**, **Cursor**, **Codex**, **Windsurf**, **Gemini CLI**, and [40+ agents](https://agentskills.io).

## Install

**CLI (recommended):**

```bash
npx skills add andginja/marketingskills
```

**Claude Code plugin:**

```bash
/plugin marketplace add andginja/marketingskills
```

**Git clone:**

```bash
git clone https://github.com/andginja/marketingskills.git .agents/skills/marketingskills
```

**Git submodule:**

```bash
git submodule add https://github.com/andginja/marketingskills.git .agents/skills/marketingskills
```

## What Are Agent Skills?

Skills are markdown files that give AI agents specialized knowledge and workflows for specific tasks. They follow the open [Agent Skills specification](https://agentskills.io) supported by 40+ AI tools.

When you install these skills, your AI agent gains expert-level marketing knowledge. Ask it to "optimize this landing page" or "write a cold email sequence" and it will use the relevant skill automatically.
